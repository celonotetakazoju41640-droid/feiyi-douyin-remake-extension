# 2026-07-15 商品图分析反馈准确性

## 用户原始需求摘要

用户要的是靠谱的一条龙，不只是功能接上，还要页面反馈说真话。

## AI 对需求的理解

当前商品图视觉分析已经接上了真实图片分析，但如果服务失败，页面最后仍然会提示“已根据商品图内容自动提炼”，容易让用户误以为这次真识别成功了。

这会直接影响用户判断结果可信度。

## 当前假设

- 本轮不改识别能力本身，只改前台反馈准确性。
- 页面需要明确区分两种情况：
  - 真实视觉分析成功
  - 已退回文件名 / 模板兜底

## 计划修改范围

- `src/workspace.js`
- `tests/extension-shell.test.mjs`
- `README.md`
- 本执行计划文档

## 验收标准

- 服务成功时，页面明确提示是按商品图内容提炼。
- 服务失败时，页面明确提示是按文件名 / 模板兜底。
- `node --check src/workspace.js`
- `node --test tests/extension-shell.test.mjs`
- `git diff --check`
- `git diff --cached --check`

## 风险和需要注意的事项
