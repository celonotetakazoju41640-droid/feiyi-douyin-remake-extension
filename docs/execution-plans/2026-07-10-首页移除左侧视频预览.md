# 2026-07-10 首页移除左侧视频预览

## 用户原始需求摘要

用户要求：

- 把首页左侧的视频预览删掉
- 把左侧栏目原来的黑色底也调掉

## 我对本次任务的理解

这轮只处理首页可见布局和配色：

- 左侧参考视频预览不再展示
- 右侧上传和设置区占满首页主工作区
- 左侧栏目从黑底改成浅色暖底
- 现有视频上传逻辑继续保留

## 当前假设

- 用户只是不想在首页看到左侧预览，不是要删除参考视频上传能力
- 这轮主要改首页生成页的 HTML 和 CSS
- 视频预览相关 DOM 节点可以保留在隐藏区，避免现有 JS 失效

## 计划修改范围

- `src/workspace.html`
- `src/workspace.css`
- `README.md`
- 本执行计划文档

## 明确不修改的范围

- `src/workspace.js`
- `src/remake-core.js`
- 历史记录
- 蒸馏管理
- 生成逻辑

## 执行步骤

1. 删除首页左侧可见预览区
2. 把预览节点移到隐藏区保留给现有脚本使用
3. 让右侧内容区占满
4. 把左侧栏目黑底改成浅色暖底
5. 跑检查并核对真实页面

## 验收标准

- 首页不再显示左侧视频预览
- 左侧栏目不再是黑色底
- 视频上传功能不报错
- 右侧上传和设置区正常显示
- `git diff --check` 通过
