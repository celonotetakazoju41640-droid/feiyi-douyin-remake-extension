# 2026-07-13 TikTok 提示词英文修复

## 用户原始需求摘要

用户确认修复当前生成链路里的语言问题：TikTok 生成结果正文必须是英文，抖音继续保持中文。

## 我对本次任务的理解

这不是 UI 问题，也不是提交链路问题。  
当前平台切换、模板切换、口播语言元数据已经联动成功，但真正生成出来的批量提示词正文仍然固定是中文。  
本轮只修“提示词正文语言跟平台走”这一处。

## 当前假设

- `生成 -> 历史记录 -> 提交生成` 主链路已经可用
- 问题主要在 `src/remake-core.js` 的提示词拼装逻辑
- 现有测试入口 `tests/remake-core.test.mjs` 可作为本轮最小验证入口

## 计划修改范围

- `tests/remake-core.test.mjs`
- `src/remake-core.js`
- `README.md`
- 本执行计划文档

## 明确不修改的范围

- 不改 UI 结构
- 不改本地服务接口
- 不改蒸馏管理逻辑
- 不改深蒸馏分析逻辑
- 不顺手调整历史记录页和生成页样式

## 执行步骤

1. 先补一条失败测试，证明 TikTok 批量提示词正文现在没有切成英文
2. 只改提示词拼装逻辑，让 TikTok 输出英文正文、抖音输出中文正文
3. 跑最小测试和生成链路复验
4. 同步 README 和计划结果

## 验收标准

- TikTok 生成出的批量提示词正文为英文
- 抖音生成出的批量提示词正文继续为中文

- 真实页面复验中，`生成 -> 历史记录` 仍然正常
- 真实页面复验中，`提交生成` 仍然正常

## 风险和需要用户确认的事项

- 如果后续要把镜头提示、蒸馏摘要、批量任务文案全部做双语优化，需要另开一轮
