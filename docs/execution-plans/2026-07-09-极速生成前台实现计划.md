# 极速生成前台 Implementation Plan

> 这份计划对应的是“先把首页压成普通用户前台”的阶段。当前已经继续收口到向导式创建任务流程，实际执行请以 `docs/execution-plans/2026-07-09-向导式UI重构-spec.md` 和 `docs/execution-plans/2026-07-09-按spec收口执行计划.md` 为准。

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把当前工作台重构成真正面向普通用户的极速生成前台，并把模板蒸馏等复杂能力后置到官方后台区域。

**Architecture:** 保留现有扩展和前端本地能力，不新增后端。首页只暴露用户最少输入，生成时自动准备模板和风格参考；模板蒸馏、样本筛片、本地服务等能力继续保留在后台折叠区。

**Tech Stack:** Chrome Extension MV3, 原生 HTML/CSS/JS, 本地存储, 现有 `remake-core.js` 生成链路
