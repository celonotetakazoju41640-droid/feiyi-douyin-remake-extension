# 2026-07-10 首页设置框内部空白收紧

## 用户原始需求摘要

用户要求把首页“中文视频设置”这条里每个框内部多余的空白优化掉，不要显得这么空。

## 我对本次任务的理解

这轮只收紧设置条里各个输入框和下拉框的内部空白，不改字段、不改流程、不改其他区块布局。

## 当前假设

- 当前空白主要来自字段列宽偏大
- 输入框和下拉框的内边距也略宽
- 只改样式即可完成，不需要改 HTML 和 JS

## 计划修改范围

- `src/workspace.css`
- 本执行计划文档

## 明确不修改的范围

- `src/workspace.html`
- `src/workspace.js`
- 历史记录
- 蒸馏管理
- 字段文案和字段顺序

## 执行步骤

1. 收紧设置条内各列宽度
2. 微调输入框和下拉框的内边距
3. 打开真实页面确认框内空白减少
4. 跑最小检查并提交

## 验收标准

- 用户圈出的几个框内部空白明显减少
- 各字段仍保持一条横向对齐
- 桌面端不出现挤压错位
- `git diff --check` 通过

## 执行后的实际结果

- 已收紧设置条 6 个字段的列宽
- 已微调下拉框和输入框的内边距
- 用户圈出的框体内部空白明显减少，整体更紧凑

## 验证方式

- 打开本地 `src/workspace.html` 查看真实页面
- 人工核对设置条里各输入框和下拉框的内部空白是否明显减少
- 补跑：
  - `git diff --check`
