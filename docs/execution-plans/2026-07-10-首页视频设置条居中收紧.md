# 2026-07-10 首页视频设置条居中收紧

## 用户原始需求摘要

用户要求把“中文视频设置”这一条里的框整体居中，并减少多余留白。

## 我对本次任务的理解

这轮只调整首页视频设置条的排版密度和对齐方式，不改字段、不改功能、不改其他页面结构。

## 当前假设

- 当前问题主要来自设置条网格列宽过宽
- 每个字段块本身也缺少统一的居中约束
- 只改样式即可完成，不需要改 JS 和 HTML 结构

## 计划修改范围

- `src/workspace.css`
- 本执行计划文档

## 明确不修改的范围

- `src/workspace.html`
- `src/workspace.js`
- 历史记录
- 蒸馏管理
- 字段文案和字段数量

## 执行步骤

1. 检查设置条当前网格和字段块样式
2. 收紧列宽与间距，让整条内容居中
3. 打开真实页面确认左右留白减少且控件对齐
4. 跑最小检查并提交

## 验收标准

- “中文视频设置”这一条中的 6 个控件整体更靠中
- 左右多余留白明显减少
- 各字段上下对齐，不出现挤压错位
- `git diff --check` 通过

## 执行后的实际结果

- 已将设置条改为按内容宽度居中显示
- 已收紧 6 个控件的列宽与间距
- 桌面端右侧留白明显减少，整体更像一整条紧凑设置区

## 验证方式

- 打开本地 `src/workspace.html` 查看真实页面
- 人工核对“中文视频设置”条在桌面端是否居中且更紧凑
- 补跑：
  - `git diff --check`
  - `git status --short --branch`
