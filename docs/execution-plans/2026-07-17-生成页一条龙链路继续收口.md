# 2026-07-17 生成页一条龙链路继续收口

## 用户原始需求

继续把 Chrome 扩展工作台里“选择蒸馏模型 -> 传图 -> AI 自动识别产品卖点和场景 -> 自动生成提示词 -> 一键生成项目 / 故事版 -> 复制提示词和图片带走”这条链路收完整，前台尽量傻瓜式，复杂逻辑藏后台。

## 本轮目标

只收口 `飞蚁抖音复刻` 这版生成页主链路，让用户在生成页主屏就能看懂：

1. 当前选的是哪条模板 / 模型链路。
2. 商品图识别到了什么。
3. 自动补出来的提示词草稿是什么状态。
4. 下一步应该直接生成，还是先等识别，还是去历史记录带走结果。
