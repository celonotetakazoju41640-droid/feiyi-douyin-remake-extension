# 2026-07-15 向导生成按钮不再绕过上传门槛

## 用户原始需求摘要

用户要的是整条主链路口径一致，不能主按钮禁了，另一个入口却还能偷偷生成。

## AI 对需求的理解

代码审计发现：

- 主生成按钮已经要求“当前会话里必须重新上传商品图”
- 但向导区底部的 `生成并带走` 按钮还没有同步这条门槛

这会导致：

- 历史恢复后
- 主按钮看起来不能生成
- 用户仍可能从向导按钮绕过门槛触发一次新的生成

## 当前假设

- 本轮只修向导按钮和主按钮门槛一致性。
- 不改生成逻辑本身，不改 UI 结构。

## 执行步骤

1. 补测试，锁定向导按钮不能绕过 fresh upload 门槛。
2. 修改 `renderWizardStep()`，让第 3 步向导按钮跟主按钮共用当前输入可生成判断。
3. 修改 `handleWizardNext()`，即使按钮状态异常，也继续在逻辑层兜底拦一次。
4. 同步 `README.md`。
5. 跑检查命令并完成 Git 闭环。
