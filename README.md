# 飞蚁抖音复刻

这是一个 Chrome 扩展版的短视频复刻工作台，目标不是一键直出成片，而是先把参考短视频拆成可执行的复刻结构，再整理成提示词、镜头表和批量任务。

当前仓库重点解决 4 件事：

1. 沉淀多个对标账户模板，而不是每次从零写提示词。
2. 支持抓取 TikTok 公开主页样本，做账号蒸馏和样本筛片。
3. 把复刻结果整理成复刻包、Clipcat 指令、批量视频任务和导出文件。
4. 给 TikTok / 抖音双平台打通统一入口，并支持公开主页样本蒸馏。

如果你是刚接手这个仓库，建议先看这份说明，再看路线文档：

- [ROADMAP.md](./ROADMAP.md)

## 平台边界

### TikTok

- 支持公开主页样本自动蒸馏。
- 支持基于主页样本做筛片、重蒸馏和摘要回填。
- 需要扩展拥有 `https://www.tiktok.com/*` 页面权限。

### 抖音

- 已支持公开主页样本级别的自动蒸馏。
- 当前已从最小可用版推进到第二层增强：抓公开主页样本、回填模板、优先补标题 / 封面 / 时长 / 互动数据，并复用现有筛片台。
- 在真实可见卡片存在时，当前还能直接补到：
  - 卡片封面图
  - 点赞数
  - 去重后的标题文案
- 还不支持登录态抓取、私有内容和深翻账号详情。

## 安装方式

1. 打开 Chrome。
2. 进入 `chrome://extensions/`。
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择当前目录：
   - `/Users/da/Desktop/飞蚁抖音复刻-Chrome扩展`

加载后会出现扩展：

- 名称：`飞蚁抖音复刻`
- 弹窗入口：`src/popup.html`
- 工作台页面：`src/workspace.html`

## 使用流程

### 1. 打开工作台

从扩展弹窗点击“打开复刻工作台”。

### 2. 上传参考短视频

- 优先上传本地参考视频。
- 参考视频用于拆钩子、节奏、镜头功能和转化结构。
- 不建议把它当成直接照抄的成片模板。

### 3. 上传商品图

- 商品图是必填。
- 当前最多 9 张。
- 这一步是为了把结构复用到你的商品，而不是只复用原视频剧情。

### 4. 选择或建立对标模板

你可以：

- 直接选已有模板。
- 新建空模板。
- 手动填写模板字段。
- 如果是 TikTok 账号主页，可以直接做公开样本蒸馏。

### 5. TikTok 主页蒸馏

如果当前模板平台是 TikTok：

- 填主页链接。
- 点击“扫描 TikTok 主页并蒸馏”。
- 在样本区做筛片。
- 必要时“按选中样本重蒸馏”。
- 最后把蒸馏摘要写入参考区。

如果当前模板平台是抖音：

- 可直接填写抖音公开主页链接。
- 点击“扫描抖音主页并蒸馏”。
- 当前先做公开样本抽取，不登录、不抓私有内容。

### 6. 生成复刻包

补齐：

- 商品名
- 产品补充信息
- 钩子风格
- 视觉风格
- 收口引导

然后点击“拆解复刻”。

结果区会给出：

- 项目统计
- 复刻结果
- 可下载 JSON
- 可下载 Markdown
- 可复制批量任务
- 可发送到本地服务

### 7. Clipcat 模式

如果你要把结果转到外部复刻工具，可以直接在工作台里补：

- 参考平台
- 参考短视频链接
- 口播语言
- 附加规则

然后点击“复制 Clipcat 指令”。

## 权限说明

当前扩展权限来自 `manifest.json`：

- `activeTab`
- `downloads`
- `scripting`

当前页面权限：

- `https://www.tiktok.com/*`
- `https://www.douyin.com/*`
- `http://127.0.0.1:4328/*`
- `http://localhost:4328/*`

这意味着：

- 可以在 TikTok / 抖音页面上执行公开样本抓取相关逻辑。
- 可以访问本地 `4328` 服务。
- 当前仍不会用这些权限去抓登录态或私有内容。
