# 飞蚁抖音复刻

这是一个 Chrome 扩展版的短视频复刻工作台，目标不是一键直出成片，而是先把参考短视频拆成可执行的复刻结构，再整理成提示词、镜头表和批量任务。

当前仓库重点解决 4 件事：

1. 沉淀多个对标账户模板，而不是每次从零写提示词。
2. 支持抓取 TikTok 公开主页样本，做账号蒸馏和样本筛片。
3. 把复刻结果整理成复刻包、Clipcat 指令、批量视频任务和导出文件。
4. 给 TikTok / 抖音双平台打通统一入口，并支持公开主页样本蒸馏。

如果你是刚接手这个仓库，建议先看这份说明，再看路线文档：

- [ROADMAP.md](./ROADMAP.md)

## 平台边界

### TikTok

- 支持公开主页样本自动蒸馏。
- 支持基于主页样本做筛片、重蒸馏和摘要回填。
- 需要扩展拥有 `https://www.tiktok.com/*` 页面权限。

### 抖音

- 已支持公开主页样本级别的自动蒸馏。
- 当前已从最小可用版推进到第二层增强：抓公开主页样本、回填模板、优先补标题 / 封面 / 时长 / 互动数据，并复用现有筛片台。
- 在真实可见卡片存在时，当前还能直接补到：
  - 卡片封面图
  - 点赞数
  - 去重后的标题文案
- 还不支持登录态抓取、私有内容和深翻账号详情。

## 使用流程

### 1. 打开工作台

从扩展弹窗点击“打开复刻工作台”。

### 2. 填主页链接生成模板

- 先选 TikTok 或抖音。
- 填对标主页链接。
- 点击“提炼主页模板”。
- 扫完后会自动命名并加入模板列表。

### 3. 上传模特图和产品图

- 模特图可选，用于固定人物风格。
- 产品图必填，当前最多 9 张。
- 这一步是为了把结构复用到你的商品，而不是只复用原视频剧情。

### 4. 填提示词和生成要求

- 建议填写：
  - 商品名
  - 创作提示词
  - 生成条数
- 商品补充要求可以不填。
- TikTok 默认英文提示词，抖音默认中文提示词。

### 5. 生成提示词

- 点击“生成提示词”后，结果区会保留：
  - 最近生成项目
  - 第一条提示词预览
  - JSON / Markdown 下载

- 更细的镜头表、批量任务、本地服务入口都在二级折叠区，不会默认打开展示。

### 6. 高级区

如果你要把结果转到外部复刻工具，可以直接在工作台里补：

- 参考平台
- 参考短视频链接
- 口播语言
- 附加规则

然后点击“复制 Clipcat 指令”。

- 负责整理任务

## 权限说明

当前扩展权限来自 `manifest.json`：

- `activeTab`
- `downloads`
- `scripting`
- `storage`

当前页面权限：

- `https://www.tiktok.com/*`
- `https://www.douyin.com/*`
- `https://vip.gptsdd.com/*`
- `http://127.0.0.1:4328/*`
- `http://localhost:4328/*`

这意味着：

- 可以在 TikTok / 抖音页面上执行公开样本抓取相关逻辑。
- 旧的后台消息转发链路现在会正式挂到 MV3 service worker，不再只是代码在仓库里但实际没启动。
- 如果需要走 GPTSDD 页面自动化，当前扩展有最小必要站点权限去打开和注入该页面。
- 可以访问本地 `4328` 服务。
- 当前仍不会用这些权限去抓登录态或私有内容。
